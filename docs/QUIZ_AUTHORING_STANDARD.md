# Quiz Authoring Standard

This standard applies to every multiple-choice quiz in the platform.

## Principles

1. All four options should be similar in length.
2. Distractors should be historically plausible.
3. Correct answers should be evenly distributed across A, B, C and D.
4. Questions should increase in cognitive demand through the quiz:
   - Recall
   - Understanding
   - Explanation
   - Judgement/application
5. Vocabulary should reflect AQA A-Level History terminology.
6. There must be one clearly best answer, but the distractors should require genuine thinking.

## Author QA checklist

Before committing a quiz, check:

- [ ] No option stands out because it is much longer or shorter.
- [ ] Distractors represent realistic misconceptions.
- [ ] Correct answer positions are balanced.
- [ ] Wording is concise and precise.
- [ ] Questions reinforce exam language.
- [ ] The quiz includes a mix of recall and higher-order thinking.

## Future improvements

Existing quizzes should be updated to meet this standard whenever they are revisited. New quizzes must comply before they are merged.
